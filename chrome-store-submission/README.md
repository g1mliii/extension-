# URL Trust Rater - Browser Extension

A community-driven browser extension that allows users to rate websites and view aggregated trust scores to make informed browsing decisions.

## Features

- **Trust Score System**: View community-driven trust scores (0-100) for any website
- **Rate Websites**: Submit 1-5 star ratings for websites you visit
- **Report Content**: Flag websites as spam, misleading, or scam content
- **Real-time Feedback**: See immediate impact of your ratings with local calculations
- **Security Integration**: Built-in recommendations for security tools
- **iOS 26 Design**: Modern liquid glass UI with smooth animations

## Installation

### From Chrome Web Store (Recommended)
1. Visit the Chrome Web Store (link coming soon)
2. Click "Add to Chrome"
3. Confirm installation

### Manual Installation (Development)
1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The extension icon will appear in your browser toolbar

## Usage

1. **View Trust Scores**: Click the extension icon on any website to see its trust score
2. **Rate Websites**: Use the star rating system to rate your experience
3. **Report Issues**: Use flag buttons to report spam, misleading, or scam content
4. **Account Required**: Create a free account to submit ratings and reports

## Privacy & Security

- **No Personal Data Collection**: We only collect ratings and website URLs
